---
name: powerlines-deploy
description: Use when the user wants to deploy a Powerlines project and its generated artifacts.
---

Run the Powerlines `deploy` command to publish deployable outputs.

## When To Use

- User asks to deploy with Powerlines.
- User asks to publish generated assets.

## Command

```bash
pnpm exec powerlines deploy --root <project-root>
```

Aliases are also available: `deployment`, `publish`.
