---
name: powerlines-create
description: Use when the user wants to initialize a new Powerlines project.
---

Run the Powerlines `create` command to initialize a project scaffold.

## When To Use

- User asks to create a new Powerlines project.
- User asks to initialize Powerlines in a specific directory.

## Command

```bash
pnpm exec powerlines create --root <project-root>
```

`root` is required for this command.
